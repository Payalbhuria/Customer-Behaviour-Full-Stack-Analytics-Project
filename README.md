📊  Customer Behaviour Full-Stack Analytics Project
🔍 Overview

The goal of this project is to simulate a corporate-grade end-to-end data analytics workflow, demonstrating the ability to translate raw data into strategic business intelligence by:

✅ Data Preparation,Modeling & Exploratory Data Analysis (Python): Clean and transform the raw dataset for analysis.

✅ Data Analysis (SQL): Simulate business transactions, and run queries to extract insights on customer segments, loyalty, and purchase drivers.

✅ Visualization & Insights (Power BI): Build an interactive dashboard that highlights key patterns and trends, enabling stakeholders to make data-driven decisions.
⚙️ Project Steps
1. Data Loading
Imported dataset using Python (Pandas)
Verified structure and data types
2. Data Cleaning
Handled missing values (e.g., review ratings)
Standardized column names
Removed inconsistencies and duplicates
3. Exploratory Data Analysis (EDA)
Analyzed customer demographics
Studied purchase patterns
Identified trends across categories and locations
4. SQL Integration (PostgreSQL)
Created database and tables
Loaded cleaned data into PostgreSQL
Executed SQL queries for:
Revenue analysis
Top customers
Category performance
Payment method distribution
5. Dashboard Development (Power BI)
Built interactive dashboard with:
KPI cards (Total Customers, Revenue, Ratings)
Category-wise sales analysis
Payment method distribution
Customer-level insights
Added slicers for dynamic filtering
📊 Dashboard

The Power BI dashboard provides a visual representation of key insights:

Revenue by category
Number of purchases by category
Payment method usage
Customer segmentation
Top customers analysis
📈 Results & Insights
Identified high-performing product categories
Analyzed customer spending behavior across demographics
Observed dominant payment methods
Highlighted top customers contributing to revenue
Provided actionable insights for business decision-making
